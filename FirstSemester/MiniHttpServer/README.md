# Установка и запуск

## 1. Предварительные требования

Установленные:

.NET SDK.

PostgreSQL.

## 2. Настройка Базы Данных
Создайте базу данных в PostgreSQL (например, `tours_db`).
Выполните SQL-скрипт инициализации (создание таблиц и начальных данных). Скрипт находится в файле `init_db.sql`

## 3. Конфигурация
Откройте файл `Settings/settings.json` и укажите свои данные для подключения:
`"Domain": "localhost",`

`"Port": 1234,`

`"ConnectionString": "Host=localhost;Port=5432;Database=tours_db;Username=postgres;Password=your_password",`

## 4. Запуск приложения (Локально)
Откройте терминал в папке проекта(MiniHttpServer) и выполните:
### 1. Восстановление зависимостей
`dotnet restore`
### 2. Сборка и запуск
`dotnet run`

Сервер запустится по адресу: http://localhost:1234/

Список всех туров можно посмотреть на странице `/tours/list`

## 5. Остановка сервера
Для остановки сервера выполните следующую команду:

`/stop`
